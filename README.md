Project Budget Generator
Description
This tool is a specialized script designed for freelancers and project managers. It is an automated solution to create professional PDF documents from terminal inputs.
Requirements
To run this project, you need:
Python 3.x installed.
fpdf library: Used for PDF generation.
A Template.png file in the root folder to act as the background.
Installation:
pip install fpdf
How it works
The script starts by displaying a welcome banner with the current date.
The user enters the project details such as name, hours, and costs.
The logic calculates the total budget using precise mathematical operations.
It creates a budgets/ folder and saves a customized PDF using a background template.
Finally, it asks the user if they want to generate another file or exit.
